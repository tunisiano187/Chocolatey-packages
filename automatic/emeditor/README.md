## EmEditor

EmEditor is a fast, lightweight, yet extendable, easy-to-use text editor for Windows. Both native 64-bit and 32-bit builds are available!

This is a 30 day trial version but you can use free version past its day.

#### [choco://emeditor](choco://emeditor)
To use choco:// protocol URLs, install [(unofficial) choco:// Protocol support ](https://chocolatey.org/packages/choco-protocol-support)
